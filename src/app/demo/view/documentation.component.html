GUIDE RAPIDE D'UTILISATION DE ADOBLOCK

Basé sur Angular 15, PrimeNG 15, Spring boot 3 et Ethereum 2

1- Obtenir une paire de clés cryptographiques

Cette action est uniquement réservée aux entités administratives, qui souhaitent enregistrer des documents dans la blockchain Ethereum pour des besoins d'authentification ultérieure.

2- Enregistrer un document administratif sur Ethereum

Cette action est uniquement réservée aux auteurs ou signataires de documents administratifs. Pour l'effectuer, il faut :

  1. Générer ou obtenir une paire de clés cryptographiques
  2. Renseigner la clé publique obtenue dans le document qui est prévu pour être stocker sur Ethereum
  3. Charger la version finale du document administratif au format PDF ou WORD. Renseigner en même temps, les deux (02) clés
  4. Demander (en validant) la transaction de stockage dudit document sur Ethereum
  5. Prendre connaissance de la réponse de la blockchain

3- Vérifier l'authenticité d'un document administratif sur Ethereum

Cette action est destinée à tout le monde (entités administratives, usagers-clients). Pour l'effectuer, il faut :

  1. Charger la version finale du document administratif au format PDF ou WORD
  2. Demander (en validant) la transaction d'authentification dudit document sur Ethereum
  3. Prendre connaissance de la réponse de la blockchain

4- Gérer le compte utilisateur

Cette action est destinée aux entités administratives
